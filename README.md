# 🚀 Projetos de API REST com Python

Este repositório reúne os projetos que venho desenvolvendo ao longo dos meus estudos em **Python** com foco na criação de **APIs REST**, utilizando os conteúdos aprendidos na plataforma da DIO (Digital Innovation One).

O objetivo principal é consolidar conhecimentos práticos em desenvolvimento backend, organização de código e boas práticas na construção de APIs.

---

## 📚 Sobre os estudos

Durante o bootcamp, estou explorando conceitos importantes como:

- Criação de APIs com frameworks modernos (ex: FastAPI)
- Estruturação de projetos backend
- Boas práticas com rotas, controllers e models
- Uso de parâmetros (path e query)
- Integração com bancos de dados
- Versionamento de código com Git e GitHub

---

## 🛠️ Tecnologias utilizadas

- Python 🐍  
- FastAPI ⚡  
- Uvicorn  
- SQLite / PostgreSQL (dependendo do projeto)  
- Git & GitHub
